# 📊 E-Commerce Sales Analysis (Power BI)

## 📌 Project Overview  
This project presents an end-to-end analysis of e-commerce sales data using **Power BI**, transforming raw datasets into interactive dashboards that provide actionable business insights.

The dashboard enables stakeholders to monitor key performance metrics, identify trends, and make data-driven decisions related to sales, profitability, and operations.

---

## 🎯 Business Problem  
E-commerce businesses often lack clear visibility into:
- Sales performance  
- Product demand  
- Regional profitability  
- Impact of returns on revenue  

This project addresses these challenges by delivering an interactive analytics solution.

---

## 🛠️ Tools & Technologies  
- Power BI Desktop  
- Power Query  
- DAX (Data Analysis Expressions)  
- CSV Data Sources  

---

## 📂 Dataset Description  
The project uses multiple datasets including:
- Amazon Sales Report  
- International Sales Report  
- Monthly Sales Data  
- Profit & Loss Data  
- Expense Data  

---

## 🔄 Data Preparation  
- Cleaned and transformed raw data using Power Query  
- Removed duplicates and handled missing values  
- Standardized formats across datasets  
- Built relationships between tables  
- Created calculated measures using DAX  

---

## 📊 Key KPIs  
- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin (%)  
- Return Rate (%)  

---

## 📊 Dashboard Pages  

### 1️⃣ Sales Overview  
- Sales, Profit, and Orders summary  
- Monthly sales trends  
- Category-wise performance  

### 2️⃣ Product Analysis  
- Top 10 products by sales  
- Category performance  

### 3️⃣ Regional Analysis  
- Sales distribution by region  
- Profit comparison across regions  

### 4️⃣ Returns & Profitability  
- Return rate analysis  
- Impact of returns on profit
