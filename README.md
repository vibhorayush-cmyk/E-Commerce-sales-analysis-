# 📊 E-Commerce Sales Analysis (Power BI Project)

## 📌 Project Overview  
This project focuses on analyzing e-commerce sales data using Power BI to uncover key business insights related to revenue, customer behavior, product performance, and regional trends.

---

## 🎯 Business Objectives  
- Identify top-performing products and categories  
- Analyze sales trends over time  
- Track regional performance and customer demand  
- Monitor return rates and profitability  

---

## 📂 Dataset Description  
Datasets used:
- Amazon Sale Report  
- International Sales Report  
- Monthly Sales Data  
- Profit & Loss Data  
- Expense Data  

---

## 🛠️ Tools & Technologies  
- Power BI  
- Power Query  
- DAX
